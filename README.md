# SportNest

## Purpose

SportNest is a sports facility booking platform where users can browse, book, add, update, and manage sports facilities.

## Live URL

Client: your-live-client-url
Server: your-live-server-url

## Features

- Browse all facilities
- Search and filter facilities
- Book facilities
- Add/manage own facilities
- Cancel bookings
- Better Auth login/register/google auth
